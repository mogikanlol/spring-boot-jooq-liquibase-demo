## Plugins

* [Liquibase Gradle Plugin](https://github.com/liquibase/liquibase-gradle-plugin)

## Liquibase
### Migrate
```console
gradlew.bat update
```

### Rollback
```console
gradlew.bat rollbackCount -PliquibaseCommandValue=1
```
